About

This project was inspired by my day-to-day job. I use many tools and services at work and often need to generate strong passwords when signing up, be able to remember them when logging in, and access them when I'm working away from my usual desktop. This necessity drove me to create PassGen, a secure and easy-to-use password generator and manager.

My goal was to create a tool that helps users generate strong passwords and store them safely. This project was developed as a portfolio project for ALX SE foundations.
